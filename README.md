# ADBGUI
ABDGUI (ADB stands for Android Debugging Bridge) is a tool to remove pre-installed apps in your android phone.

## Stack
- [Svelte](https://vuejs.org/)
- [Tailwind](https://tailwindcss.com/)
- [Tauri](https://tauri.studio/en/)
- and many others!
